# TUWT(Tizi User Withdraw Token)

Tizi User Withdraw Token (TUWT) is a NFT that is minted when a withdrawal&#x20;request has been made, but the main fund vault does not have sufficient&#x20;amount of USDC to cover the request. This should not happen very often, as&#x20;some percentage of fund is reserved for withdrawal purpose. However, when&#x20;a bridge is temporally down or a withdrawal rush happens, there might be&#x20;situations that the majority of fund is locked up in other chains / yield pool.&#x20;Usually, the fund will be recovered in 24h or depending on the protocols the&#x20;fund is in in other pool. In this circumstance, an NFT called “TUWT” will&#x20;be issued and the participants will be put in a queue based on their request&#x20;time. Once the fund vault is sufficient, the TUWT can be burnt to finalise the&#x20;withdrawal.

TUWT can be traded on NFT markets to realise its value at any time.
